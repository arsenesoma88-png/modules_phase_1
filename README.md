# modules_phase_1
Projets réalisés dans le cadre des modules de formation en python

📘 Formation Python – Phase 1
Ce dépôt contient les ressources et exercices de la formation Python (Phase 1).
Il est organisé en 7 modules progressifs, allant des bases du langage jusqu’à l’utilisation de Git et du terminal.

📂 Structure des modules
Module 1 – Premiers pas en Python  
Introduction au langage, installation, premiers scripts et notions fondamentales.

Module 2 – Opérateurs et conditions  
Utilisation des opérateurs arithmétiques, logiques et mise en place des conditions if/elif/else.

Module 3 – Les boucles  
Découverte des boucles for et while, contrôle du flux et itérations.

Module 4 – Fonctions  
Définition et utilisation de fonctions, paramètres, retour de valeurs et bonnes pratiques.

Module 5 – Listes, tuples et dictionnaires  
Structures de données essentielles, manipulation et parcours.

Module 6 – Les fichiers en Python  
Lecture et écriture de fichiers (open, read, write), gestion des chemins, bonnes pratiques pour manipuler les données persistantes.

Module 7 – Git, GitHub et terminal  
Gestion de versions, création de dépôts, commandes essentielles du terminal, collaboration via GitHub.

🚀 Objectifs pédagogiques
Acquérir les bases solides du langage Python.

Comprendre et manipuler les principales structures de données.

Développer des scripts modulaires et réutilisables.

Maîtriser la gestion des fichiers pour stocker et lire des données.

Utiliser Git et GitHub pour le versioning et la collaboration.

Être autonome dans l’utilisation du terminal et l’organisation de projets.

🛠️ Prérequis
Installation de Python 3.x

Éditeur recommandé : Thonny, VS Code ou PyCharm

Compte GitHub pour la gestion des dépôts

📖 Utilisation
Clonez le dépôt :

bash
git clone https://github.com/arsenesoma88-png/modules_phase_1.git
Naviguez dans le module souhaité :
